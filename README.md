
# Koraq Web Showcase

A simple website for the Koraq app, a daily profit tracker for WhatsApp sellers, POS agents, and informal business owners in Nigeria.

## Features

- **Dashboard** — Today's sales, expenses, net profit, weekly chart
- **Add Transaction** — Sale or Expense details (amount, category, note, date)
- **History** — Daily transaction list, weekly & monthly profit summaries
- **Reports** — Monthly overview, sales-vs-expenses chart, WhatsApp sharing
- **Offline-first** — All data stored locally in the app

## Design

Premium fintech look: dark background (navy/charcoal), emerald green for profit, soft red for loss, electric blue accent. Big numbers, minimal text, smooth animations.

## Website Purpose

This site showcases Koraq's features and design. All actions prompt users to download the app. No data is stored or actions performed on the website.
